🐕 RAG - Cão Radar
Sistema de perguntas e respostas baseado em RAG (Retrieval-Augmented Generation) desenvolvido como atividade prática, utilizando o TCC "Cão Radar" como base de conhecimento.

📌 Problema
Dificuldade em consultar e extrair informações relevantes de documentos acadêmicos longos.

💡 Solução
Pipeline RAG que combina busca vetorial (ChromaDB + Sentence Transformers) com geração de respostas via LLM (Groq - LLaMA 3.1).

🛠️ Tecnologias utilizadas
Python
ChromaDB
Sentence Transformers
Groq API (LLaMA 3.1)
Google Colab

▶️ Como executar
Abra o arquivo .ipynb no Google Colab
Importe o documento oficial do tcc cão radar que está no repositório na pasta raiz do Google Colab:
image
Execute as células em ordem
